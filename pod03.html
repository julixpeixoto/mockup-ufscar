Rest-Assured é um framework open-source que possui suporte para automatizar os métodos GET, POST, PUT, DELETE, HEAD, OPTIONS e TRACE de uma API e permitir fazer validações no header, body e schema da resposta. Restassure.io

Configurando o Ambiente


- Configurar o Java (utilizei Open JDK 1.8);
- Baixar a IDE (Eclipse ou IntelliJ);

Criar um novo projeto Maven, ou converte-lo em um projeto Maven. Isso é necessário para adicionarmos as dependências no arquivo pom.XML que será criado.

OBS: No Eclipse ou no IntelliJ, será necessário configurar o biblioteca correta do Java, pois automaticamente será utilizada a versão 1.5.

No Eclipse, acessar as propriedades do projeto > Java Build Path > aba Libraries > remover a biblioteca “J2SE 1.5” E adicionar a JRE do sistema “JRE System LIbrary” > next > ele irá exibir a versão correta > “Apply and close”
